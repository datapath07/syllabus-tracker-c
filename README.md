# Syllabus Tracker (C Project)

This is my first C project using file handling and structures.  
It helps track syllabus topics and their completion status.

## Features
- View complete syllabus
- Check particular topic status
- Update uncovered topics
- Stores data in binary file

## Technologies Used
- C Language
- File Handling
- Structures

## How to Run
1. Compile the program:
   gcc MAINPROJECT.c -o app
2. Run:
   ./app

## Learning Outcome
- File handling in C
- Struct and array usage
- Menu driven program
- Data persistence using binary files
